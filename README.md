# Lynex

Pixel-accurate implementation of the provided Figma design using React + Tailwind CSS, with a separate backend service.

## Project Structure

- `frontend`: React (Vite) + Tailwind UI implementation of the Figma design
- `backend`: Express API service

## Run Frontend

```bash
cd frontend
npm install
npm run dev
```

## Run Backend

```bash
cd backend
npm install
npm run dev
```

Backend health endpoint:

- `GET /api/health`
